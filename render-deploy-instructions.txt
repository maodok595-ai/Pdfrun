INSTRUCTIONS DE DÉPLOIEMENT SUR RENDER
===========================================

📋 FICHIERS NÉCESSAIRES (DÉJÀ CRÉÉS)
✅ requirements.txt - Contient toutes les dépendances
✅ app.py - Utilise port 5000 en local; Render fournit $PORT
✅ templates/index.html - Interface utilisateur

🚀 CONFIGURATION RENDER
=======================

1. BUILD COMMAND (Commande de construction) :
   pip install -r requirements.txt

2. START COMMAND (Commande de démarrage) :
   gunicorn -b 0.0.0.0:$PORT app:app

3. VARIABLES D'ENVIRONNEMENT À DÉFINIR :
   - SECRET_KEY: [générer une clé secrète forte]
   - FLASK_DEBUG: false (optionnel)
   
   NOTE IMPORTANTE : Ne PAS définir PORT ! Render fournit automatiquement 
   la variable $PORT. L'application l'utilisera automatiquement.
